# cyber-training-app
A gamified cybersecurity training web application that simulates real-world security scenarios, tracks user behavior, and updates organizational risk metrics through an interactive dashboard.

## GitHub Pages (live demo)

After you enable **Pages** in the repo settings (see below), the site will be available at:

**https://sabagnale-code.github.io/cyber-training-app/**

Static files (`index.html`, `style.css`, `script.js`) are at the **repository root**, which matches the default Pages setup for project sites.
